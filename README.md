# GRASP_Problems_Optimization
Репозиторий содержит реализацию следующих задач с помощью GRASP
* Задача коммивояжера (TSP)
* Задача о рюкзкаке (knapsack problem)
* Минимизация функций

## Быстрый старт
Пример использования содержится в test.py, запустите командой
```bash
python3 test.py
```

## Зависимости
Установите зависимости, используя pip, следующей командой
```bash 
pip install numpy scipy pandas
```
